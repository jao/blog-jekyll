I made this because I always disliked those happy IM windows, with things bouncing around making weird sound effects. It was intially much simpler than this, but a fancy terminal is still good, as long as it is fast, right? My friends Caffo, Taiza and Juca helped me with this since the beginning, and there is even a Variant (color style) made by Caffo that is in the package -  bu noone likes it. I've heard that this might work on pidgin too, can someone confirm this? If not what changes do I need to make so this can work?
